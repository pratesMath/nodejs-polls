<h1>NodeJS Polls</h1>

Start development:

```sh
npm ci
```

create the project from scratch:

```sh
npm init -y
```

dev dependencies:

```sh
npm i -D typescript @types/node tsx prisma
```

dependencies:

```sh
npm i fastify zod
```

Start TypeScript:

```sh
npx tsc --init
```

docker:

```sh
docker compose up -d
```

```sh
docker ps
```

Start Prisma (ORM):

```sh
npx prisma init
```

Start Prisma (ORM):

```sh
npx prisma migrate dev
```
